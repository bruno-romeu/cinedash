# 🎬 CineDash


> Uma plataforma moderna, minimalista e social para você organizar sua vida cinematográfica.

O **CineDash** é uma aplicação web focada na experiência do usuário, combinando o visual imersivo e escuro (Dark Mode) de plataformas de streaming com as funcionalidades sociais e de organização de trackers de filmes. 

---

## ✨ Funcionalidades

- **🌓 Modern Dark Mode:** Interface elegante e focada no conteúdo, com paleta de cores baseada em tons de zinco e índigo.
- **📚 Organização Limpa:** Adicione filmes à sua "Watchlist", marque como assistidos e dê notas.
- **🌐 Social Integrado:** Veja o que seus amigos estão assistindo e leia suas avaliações em um feed dinâmico.
- **📊 Estatísticas de Perfil:** Acompanhe seu tempo total assistido, média de notas e gêneros favoritos.
- **📱 Totalmente Responsivo:** Experiência fluida tanto no Desktop quanto no Mobile.

---

## 🛠️ Tecnologias Utilizadas

Este projeto utiliza uma stack moderna, robusta e orientada a componentes:

- **Backend:** [Laravel 11](https://laravel.com/) (PHP)
- **Frontend:** [React 18](https://react.dev/)
- **Ponte:** [Inertia.js](https://inertiajs.com/) (Sem necessidade de criar APIs RESTful manuais)
- **Estilização:** [Tailwind CSS](https://tailwindcss.com/)
- **Ícones:** [Lucide React](https://lucide.dev/)
- **Animações:** tailwindcss-animate
